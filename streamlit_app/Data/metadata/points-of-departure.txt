Points of Departure (POD) Metadata
==================================

Description
-----------

This dataset contains estimates of points of departure along with cumulative
proportions. 

Data Sources
------------

- Regulatory : Regulatory PODs derived from Aurisano et al. (2023). 

- ToxValDB : Surrogate PODs derived from Aurisano et al. (2023).

- QSAR : Predicted by the "ML2" QSAR model developed by Kvasnicka et al. 
    (2023).

Column Descriptions
-------------------

- DTXSID : The index column representing the unique chemical identifier.

- [Data Source]: The point of departure values in log10 units for a given 
    data source listed above. 

Usage Notes
-----------

The users should consider the context, limitations, and uncertainties of the 
data as detailed in the original papers. When using these data in publications 
or presentations, users should cite the original sources provided below.
